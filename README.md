# Bank Digital

Proyek backend sistem perbankan digital untuk tugas kuliah.  
Dibuat dengan Node.js + Express (rencananya).

> ⚠️ **Status: Masih dalam tahap perencanaan.**

---

## Tujuan Proyek

Membangun REST API sederhana untuk simulasi sistem bank digital, mencakup:
- Registrasi & login pengguna
- Manajemen rekening
- Transaksi (transfer, setor, tarik)
