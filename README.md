# Bank Digital

Proyek backend sistem perbankan digital untuk tugas kuliah.  
Dibuat dengan Node.js + Express (rencananya).

> ⚠️ **Status: Masih dalam tahap perencanaan.**

---
## Cara menjalankan endpoint API sistem perbankan
1-Buka VS Code kalian
2-Clone repository kami
3-Buka Terminal
4-Ketik 'npm install' 
5-Ketik 'npm run dev'
6-Lalu buka Echo API extension for VS Code
7-Run command endpoint yang ada di docs
---
## Tujuan Proyek

Membangun REST API sederhana untuk simulasi sistem bank digital, mencakup:
- Registrasi & login pengguna
- Manajemen rekening
- Transaksi (transfer, setor, tarik)
